Quiz do JP

Gamificação na Internacionalização da Educação Superior

Repositório GitHub: https://github.com/moaaskt/QuizJP

O Desafio Educacional

  • Identifica-se uma carência de engajamento em conteúdos relacionados à internacionalização.
  • Observa-se a escassez de recursos lúdicos para o ensino de culturas, idiomas e sistemas educacionais globais.
  • Há uma necessidade premente de desenvolver abordagens interativas e visualmente cativantes.

Solução: Quiz do JP

  • Apresenta-se um jogo web interativo com questões abrangendo:
    • Aspectos culturais
    • Diversidade linguística
    • Sistemas educacionais
  • A interface é configurada para operar em múltiplos idiomas (Português e Inglês).
  • O design incorpora elementos futuristas com fundos animados em 3D, utilizando a biblioteca Vanta.js.

Recursos Principais

  • Oferece um quiz interativo acompanhado de explicações educativas.
  • Implementa um sistema de pontuação em tempo real.
  • Inclui desafios bônus com curiosidades culturais adicionais.
  • Mantém um ranking global com armazenamento local de dados.
  • Integra fundos animados em 3D para uma experiência imersiva.
  • Garante compatibilidade e suporte para dispositivos móveis.

Stack Tecnológica

Tecnologia Função
React + TypeScript Estrutura principal e garantia de tipagem segura
Tailwind CSS Estilização ágil e responsiva
Vanta.js Geração de fundos animados em 3D
LocalStorage Armazenamento persistente de dados no lado do cliente
JSON Formato de dados para o banco de questões e traduções

Aspectos Distintivos do Quiz do JP

  • Caracteriza-se por um design futurista e uma experiência imersiva.
  • Possui uma interface multilíngue integrada.
  • Apresenta visualização 3D aprimorada com Vanta.js.
  • O código é modular, facilitando a expansão e manutenção.
  • Opera integralmente offline após o carregamento inicial.

Aprendizagem Lúdica e Interativa

  • Contribui para a promoção da aprendizagem sobre internacionalização.
  • Constitui uma ferramenta ideal para estudantes, docentes e instituições de ensino.
  • Sua aplicabilidade estende-se a workshops, eventos acadêmicos e à gamificação de cursos.

Próximas Etapas

  • Desenvolvimento de um backend para implementação de ranking online.
  • Exploração de modos multiplayer ou competições entre turmas.
  • Integração com Sistemas de Gestão da Aprendizagem (LMS), como Google Classroom e Moodle.
  • Capacidade de exportação de relatórios para análise pedagógica.
